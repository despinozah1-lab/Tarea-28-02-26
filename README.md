PROGRAMA DE FUNCIONES RECURSIVAS EN PYTHON
DESCRIPCIÓN

Este programa implementa diferentes funciones recursivas utilizando el lenguaje de programación Python. Incluye un menú interactivo en consola que permite al usuario ejecutar distintas operaciones matemáticas y conversiones.

El objetivo principal es demostrar el uso de la recursividad en la resolución de problemas computacionales.

CARACTERÍSTICAS

El programa permite realizar las siguientes operaciones:

Convertir un número entero a binario

Contar la cantidad de dígitos de un número

Calcular la raíz cuadrada entera de un número

Convertir números romanos a decimal

Calcular la suma de números enteros utilizando recursividad

Menú interactivo en consola

REQUISITOS

Python versión 3.x o superior

Sistema operativo Windows, Linux o macOS

Terminal o símbolo del sistema

Archivo recursividad.py

INSTALACIÓN

Descargar o copiar el archivo recursividad.py en una carpeta.

Abrir la terminal o símbolo del sistema.

Ubicarse en la carpeta donde se encuentra el archivo usando el comando cd.

Ejemplo:

cd C:\Users\TuUsuario\Desktop

EJECUCIÓN DEL PROGRAMA

Ejecutar el siguiente comando:

python recursividad.py

o

python3 recursividad.py

USO DEL PROGRAMA

Al ejecutar el programa, aparecerá el siguiente menú:

===== MENU PRINCIPAL =====

Convertir a Binario

Contar Digitos

Raiz Cuadrada Entera

Convertir a Decimal desde Romano

Suma de Numeros Enteros

Salir

El usuario debe seleccionar una opción ingresando el número correspondiente y luego ingresar los datos solicitados.

EJEMPLO DE USO

Seleccione una opcion: 1
Ingrese un numero entero: 10

Resultado:

Binario: 1010
